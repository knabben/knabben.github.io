Fv represents the future value of an operation in a particular time. The value Pv is multiplied

Fv representa o valor futuro de uma operação em determinado instante, cujo valor Presente P foi acrescido da taxa i (expressa em sua forma decimal), n vezes, de forma composta (juros sobre juros).

\begin{align} Fv = Pv * (1 + i)^n \end{align}

Present value

P representa o valor Presente da operação. O valor Futuro Fv, que é dado, deve ser descontado à taxa i (expressa em sua forma decimal) n vezes de forma composta (juros sobre juros).

\begin{align} P = \frac{Fv}{(1 + i) ^ n} \end{align}

{% highlight python %} def pv(fv, n, i): print(‘{0:.2f}‘.format(np.pv(i/100, n, 0, -fv)))

Period

n representa o tempo, P representa o valor Presente da operação e Fv, o valor Futuro; a taxa i é expressa em sua forma centesimal.

\begin{align} n = \frac{ln(\frac{Fv}{P})}{ln(1+i)} \end{align}

{% highlight python %} def nper(fv, pv, rate): print(round(np.nper(rate/100, 0, -pv, fv)))

nper(1610.51, 1000, 10) nper(4236, 3450, 2.3067) nper(600, 449.80, 5) nper(1425.76, 1000, 3)

5.0 9.0 6.0 12.0 {% endhighlight %}

Interest rates

i representa a taxa expressa em sua forma centesimal, P representa o valor Presenta da operação e Fn, o valor Futuro; n é o tempo ou período.

\begin{align} i = (\frac{Fv}{P}) ^ \frac{1}{n} \end{align}

{% highlight python %} def rate(fv, pv, n): print(‘{0:.2f}‘.format(np.rate(n, 0, -pv, fv) * 100))

Sequência uniforme de capitais postecipados

V representa o valor Presente de uma sequência uniformes postecipados R, construído em função de uma taxa i (expressa em sua forma decimal) e do tempo n vezes, de forma composta (juros sobre juros). \begin{align} V = R.\left[\frac{(1+i)^n-1}{(1+1)^n.i}\right] \end{align}

Amortização de empréstimo

Sistema de amortização francês (Price), o fato de o valor das prestações ser fixo é uma das virtudes desse método, porém os juros pagos no começo são altos, se comparados aos dos sistemas SAC e Sacre, e o valor a ser amortizado é muito pequeno.

Esse sistema é muito utilizado nos financiamentos em geral, como na compra de um carro, emprestimo pessoal, etc.

{% highlight python %}

Um amigo lhe empresta $ 1000,00 que devem ser pagos em 5 parcelas,

utilizando o Price, monte o esquema de amortização com taxa de 10% ao mês.

principal = 1000 nper = 5 per = np.arange(5) + 1

Calculo dos juros

ipmt = np.ipmt(0.1, per, 5, principal) print(ipmt)

Calculo do valor amortizado

ppmt = np.ppmt(0.1, per, 5, principal) print(ppmt) print(1000 - ppmt)

Calulo da parcela

pmt = np.pmt(0.1, 5, principal) print(pmt)
